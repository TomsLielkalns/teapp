# Create T3 App

## The stack:
- [Next.js](https://nextjs.org)
- [NextAuth.js](https://next-auth.js.org)
- [Prisma](https://prisma.io)
- [Tailwind CSS](https://tailwindcss.com)
- [tRPC](https://trpc.io)

## Deployed on:
- Vercel for CI pipeline
- PlanetScale for DB
- Clerk for user login/SSO
- Upstash for cache
- Axiom for logs
